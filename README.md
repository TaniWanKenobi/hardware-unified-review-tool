## About The Project

<div align="center">
  <img
    width="300"
    alt="CAD"
    src="assets/CAD.png"
    style="display: block; margin: 0 auto;"
  >
</div>

This integration connects **[Lapse](https://lapse.hackclub.com)** with **[Hack Club](https://hack.club)** to create timelapses of projects using tools in the Hack Club ecosystem.

The main goal is to make timelapses easy for anything you build, design, craft, or create.
